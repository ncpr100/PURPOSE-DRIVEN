Guia de Envio Passo a Passo

1Acesse business.facebook.com → WhatsApp Manager
2Ferramentas de conta → Modelos de mensagem → Criar modelo
3Categoria: Utilitario (Utility)
4Nome do modelo: pastor_welcome_khesed_tek (mesmo nome, idioma diferente)
5Idioma: Portugues (Brasil) — pt_BR
6Cabecalho → Texto → cole o texto acima
7Corpo → cole o texto acima. Insira os valores de exemplo quando solicitado para cada variavel
8Rodape → cole o rodape acima
9Botao → Chamada para Acao → Visitar site → cole texto e URL acima
10Enviar para revisao. Aprovacao: 24–48 horas.
After all 3 language approvals: Update environment variables TWILIO_WHATSAPP_TEMPLATE_SID_WELCOME_ES, _EN, and _PT in Vercel with the SIDs provided by Meta/Twilio.
After approval, test each language version in the Twilio WhatsApp Sandbox before enabling in production.
