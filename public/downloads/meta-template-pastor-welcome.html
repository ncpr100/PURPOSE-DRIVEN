Step-by-Step Submission Guide

1Go to: business.facebook.com → Your Business → WhatsApp Manager
2Navigate to: Account Tools → Message Templates → Create Template
3Category: Select Utility (account notifications, transactional messages)
4Template Name: Type pastor_welcome_khesed_tek exactly (lowercase, underscores)
5Language: Select Spanish (es)
6Header: Select Text → paste the header text above
7Body: Paste the body text above. When you type {{ Meta will prompt you to add a variable — confirm each one and provide a sample value from the table above
8Footer: Paste footer text above (optional but recommended)
9Buttons: Add CTA button → Visit Website → paste button text and URL above
10Submit: Click Submit for Review. Approval typically takes 24–48 hours
✅ Tip: While waiting for Meta approval, use the Twilio WhatsApp Sandbox for testing. See the Sandbox Setup Guide for instructions.
⚠️ After approval: Update the TWILIO_WHATSAPP_TEMPLATE_SID_WELCOME environment variable in your Vercel/Railway dashboard with the template SID provided by Meta/Twilio.
