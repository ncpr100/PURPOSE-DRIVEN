Template Submission — Agent 12 Cascade Messaging

Template 2 of 2 Category: Utility Language: Spanish (es) Prepared by Khesed-Tek Systems

This template is used by Agent 12 to send automated cascade notifications to church administrators. It replaces the previous manual notification approach and enables proactive alerts for key ministry activity.

ℹ️ What is Agent 12? Agent 12 is the Khesed-Tek AI notification agent responsible for sending cascade alerts — important platform events, ministry milestones, or system notifications that require the pastor's immediate attention.

Use Cases for This Template

New Member Registration

Notify pastor when a new visitor registers through a QR form or direct entry.

Prayer Request Alert

Forward urgent prayer requests to pastoral team for immediate response.

Donation Received

Notify administrator of significant donations or giving milestones.

System Important Alert

Platform security, subscription, or critical system notifications.

Volunteer Application

Alert ministry leaders when a new volunteer application is submitted.

Spiritual Assessment

Notify leadership when a member completes a spiritual gift assessment.

Step-by-Step Submission Guide

1Go to: business.facebook.com → Your Business → WhatsApp Manager
2Navigate to: Account Tools → Message Templates → Create Template
3Category: Select Utility
4Template Name: Type agent12_cascade_khesed_tek exactly
5Language: Select Spanish (es)
6Header: Select Text → paste the header text above
7Body: Paste body text. For sample values when prompted: use the examples from the variable reference table above
8Footer: Paste footer text
9Buttons: Add CTA → Visit Website → paste button text + URL
10Submit for Review. Approval: 24–48 hours. You will receive a notification in Meta Business Manager.
✅ After approval: Set env variable TWILIO_WHATSAPP_TEMPLATE_SID_AGENT12 with the template SID. Agent 12 will use this template SID when dispatching cascade notifications.
⚠️ Note on {{4}} character limit: Meta template bodies have a 1,024 character limit. Ensure the alertDetail value generated by the system stays under ~200 characters to leave room for the static text.
